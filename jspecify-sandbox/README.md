# jspecify-sandbox

[JSpecify](https://jspecify.dev/) を使ってnullの静的チェックを行う。

## gradle で使う

jspecifyはnullawayと連携してチェックを行うことができる。

nullawayはerrorprone経由で使う、errorproneはgradleのplugin経由で使う

- [nullaway](https://github.com/uber/nullaway)
- [errorprone](https://errorprone.info/)
- [gradle-errorprone-plugin](https://github.com/tbroyer/gradle-errorprone-plugin)
- [net.ltgt.errorprone](https://plugins.gradle.org/plugin/net.ltgt.errorprone)
